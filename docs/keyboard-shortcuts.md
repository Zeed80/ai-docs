# Горячие клавиши — AI Manufacturing Workspace

Система спроектирована по принципу **keyboard-first**: все основные ежедневные действия доступны без мыши.

---

## Глобальные

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+K` / `⌘K` | Открыть Command Palette |
| `Esc` | Закрыть палитру / модальное окно |
| `↑` / `↓` | Навигация по элементам палитры |
| `Enter` | Выбрать элемент в палитре |

### Command Palette

Позволяет быстро переходить на любую страницу, запускать AI-поиск и выполнять команды.

- Введите название страницы или документа — результаты появляются немедленно.
- `Enter` при пустом поле — запустить AI-поиск по всему рабочему месту.

---

## Чат со Светой (боковая панель)

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+K` / `⌘K` | Сфокусироваться на поле ввода |
| `Enter` | Отправить сообщение |
| `Shift+Enter` | Новая строка (без отправки) |
| `Ctrl+Enter` / `⌘Enter` | Отправить комментарий в треде |

---

## Просмотр и проверка документа (`/documents/:id/review`)

| Сочетание | Действие |
|-----------|----------|
| `A` | Одобрить документ |
| `R` | Напоминание о причине отклонения |
| `J` | Следующее поле извлечения |
| `K` | Предыдущее поле извлечения |
| `N` | Пропустить документ (перейти к следующему в очереди) |
| `Esc` | Вернуться на страницу документа |

---

## Страница документа (`/documents/:id`)

| Сочетание | Действие |
|-----------|----------|
| `Esc` | Вернуться в Inbox |

---

## Email-клиент (`/email`)

| Сочетание | Действие |
|-----------|----------|
| `N` | Создать новое письмо (режим композиции) |

|----------------------|----------|
| `Alt+перетаскивание` | Панорамирование |

## Универсальные поля ввода

| Сочетание | Действие |
|-----------|----------|
| `Enter` | Применить / подтвердить (поиск, фильтры, создание) |
| `Esc` | Отменить редактирование |

Применяется в: поиске счетов, поиске по нормативам, добавлении элементов в коллекции, создании групп чата, поиске в каталоге деталей.

---

## Советы

- **Review queue**: Используйте `J`/`K` для переключения между полями извлечённых данных, `A` для одобрения, `N` для перехода к следующему документу без решения — это позволяет просматривать несколько документов подряд без использования мыши.
- **Command Palette**: Быстрее, чем боковая навигация. `Ctrl+K` → несколько символов → `Enter`.
- **Чат**: `Ctrl+K` фокусирует поле ввода Светы из любого места на экране.
